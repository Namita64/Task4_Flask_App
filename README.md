Flask Contact Form App
📌 Project Description
This project is a simple Contact Form application built using Flask.
Users can submit their name and message through a web form.
______________
🚀 Features
•	Submit contact information
•	Display submitted entries
•	Flask routing
•	HTML templates
______________
🛠️ Technologies Used
•	Python
•	Flask
•	HTML
______________
▶️ How to Run
Install Flask:
pip install flask
Run the application:
python app.py
Open in browser:
http://127.0.0.1:5000
______________
📸 Output
The application displays submitted contact details and messages.
